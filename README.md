## Set Up

### 1. Clone this repository

```shell
git clone https://github.com/tkhquang/react-modal.git
cd react-modal
```

### 2. Install the required packages

With Yarn

```shell
yarn install
```

With NPM

```shell
npm install
```

## Run The App Locally

With Yarn

